Sobre o Ecossistema Cearense de Inovação

O Ecossistema Cearense de inovação é uma iniciativa de mapear os ecossistemas locais existentes no estado e seus componentes, com o intuito de fomentar uma maior conexão entre os participantes e com isso aumentar o desenvolvimento de inovação e novas tecnologias para a ecocômia local e nacional. Atualmente o ecossistema cearense é o terceiro maior mercado entre os ecossistemas da região, possuindo cerca de 16,6% das startups ativas e é de grande relevância para uma maior projeção desta rede, uma interrelação mais forte entre os componentes, para que se possar impulsionar ese mercado.

Como Utilizar


Visualização dos componentes do Ecossistema:

A principal função do sistema em questão é a possibilidade de visualização dos componentes do ecossistema de inovação do Ceará, todos concentrados em uma única plataforma, essa função é permitida mesmo para o usuário que não está logado no sistema.

Para facilitar a visualização dos componentes, é possível verificar as informações de cada um deles, filtrados por categoria, na barra lateral da esquerda, que conta com as categorias existentes no ecossistema e ao lado o número de componentes da categoria. Clicando em uma das categorias será aberta uma tela com cartões de apresentação referentes aos componentes do ecossistema naquela categoria, sendo possível o acesso ao site, clicando no botão “Conheça Mais”, e a localização no mapa, clicando no botão “Localização”. Desta forma é dada a possibilidade de uma visão ampla dos componentes daquela categoria especifica, facilitando a busca pelas informações necessárias.

Caso o usuário esteja procurando algum componente especifico, é possível fazer a filtragem pelo nome do componente na barra de busca que se encontra na barra de navegação da tela de exibição dos cartões.

Também é possível visualizar no mapa apenas as categorias desejadas através do botão "Filtrar", que se encontra na barra de navegação do site.

Na tela de filtragem é possível escolher as categorias que são desejadas a visualização e assim ser feita a filtragem, clicando no botão "Filtrar" no canto inferior direito da aba.

A tela que segue mostra o mapa filtrado apenas por aceleradores

Na aba comunidade, que está destacada na imagem que segue, é possível visualizar as comunidades do ecossistema e destacar a área a qual o ecossistema engloba, pressionando o botão "Ativar", assim como também é possível apagar a área destacada pressionando o botão "Desativar".

Login e Cadastro:

Para acessar a função de adicionar novos componentes e eventos, é necessário está logado no sistema, para isso basta clicar no botão “Entrar” da barra de navegação superior, digitar suas credenciais nos campos indicados e fazer o login, caso ainda não possua cadastro, é possível fazer o cadastro clicando no link “Fazer Cadastro”, no canto inferior da página de login.

Cadastro de Componentes, Eventos e Comunidades:

Para ter acesso a essa opção é necessário estar logado no sistema, quando o login for feito o botão de adicionar componentes irá dar acesso a uma tela de cadastro ao invés de redirecionar para a página de login.

Cadastro de Componentes:

Para cadastrar um novo componente do ecossistema é necessário clicar em “Cadastrar Componente”, em seguida aponte o local do endereço do componente em questão no mapa e clique em "Aqui!" para confirmar.

Após confirmar o local no mapa, irá aparecer uma tela de cadastro que deve ser preenchida com os dados do local o qual está sendo feito o cadastro e em seguida clique em “Enviar”, completando o procedimento de cadastro.

Cadastro de Eventos:

O cadastro de eventos segue o mesmo processo do cadastro de componentes até a tela de cadastro, lá é possível ver, na parte superior, a aba de eventos, onde é feito o cadastro dos próximos eventos agendados.

Os eventos podem ser verificados na página de eventos, que pode ser acessada pela barra de navegação. Na página de eventos é possível ver os próximos eventos em formato de lista, os dados dos eventos cadastrados, um botão que irá levar para o site do evento e um segundo botão que irá levar para a localização do evento o mapa.

Na barra superior existe uma barra de busca, juntamente com um menu onde pode ser feita a escolha do tipo de busca, que pode ser pelo nome do evento ou pela categoria do evento.

Cadastro de Comunidades:

O cadastro de comunidade é feito pressionando o botão "+Comunidade" da barra de navegação.

Com isso, será aberta uma aba, onde é possível adicionar os dados da comunidade, inclusive o código para que seja delimitada a área desejada. O código necessário pode ser gerado no site http://geojson.io/, bastando inserir as delimitações da região com as ferramentas do próprio site ou por meio da importação de arquivo. Após a delimitaçao, um código em JSON será fornecido, e então será necessário copia-lo e cola-lo no espaço "Coordenadas da Região", da janela mostrada na figura a baixo.
